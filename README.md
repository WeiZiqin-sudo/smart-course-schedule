# 📚 智慧课程表 - Smart Course Schedule System

一个现代化的校园课程表管理系统，支持多视图展示、冲突检测、主题切换等功能。

## ✨ 功能特性

### 核心功能
- 📅 **周视图** - 表格形式展示一周课程安排
- 📋 **列表视图** - 卡片形式展示所有课程，支持搜索、过滤、排序
- 📆 **今日视图** - 专门显示当天的课程安排

### 交互功能
- 🔍 实时搜索课程名称、教师、地点
- 🎯 按星期过滤课程
- 📊 多种排序方式（按名称、按时间）
- 🔄 周数切换（支持键盘快捷键 ← →）
- 🌓 亮色/暗色主题切换（支持 Ctrl+T）
- 📤 导出课表为文本文件
- 🖨️ 打印功能

### 视觉设计
- 🎨 现代玻璃态设计（Glassmorphism）
- 🌈 动态渐变背景
- ✨ 流畅的动画和过渡效果
- 🎯 课程冲突检测和标记
- 📱 完全响应式设计

### 用户体验
- ⏰ 实时时钟显示
- 📊 课程统计面板（课程数、教师数、课时、学分）
- 🔔 Toast 提示消息
- 📌 返回顶部按钮
- ⌨️ 键盘快捷键提示
- 🎪 骨架屏加载状态

## 🚀 快速开始

### 本地运行
1. 克隆仓库到本地
2. 在浏览器中打开 `index.html` 文件
3. 或者使用本地服务器托管文件

### 使用本地服务器
```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx serve

# 使用 PHP
php -S localhost:8080
```

## 📖 使用方法

### 视图切换
- 点击顶部标签切换不同视图
- 使用键盘快捷键快速切换

### 课程冲突提示
- 点击右上角的铃铛图标 🔔 切换冲突提示
- 冲突课程会显示粉色背景和"冲突"标签
- 点击课程查看详细的冲突信息

### 主题切换
- 点击右上角的月亮图标 🌙 切换主题
- 或使用快捷键 Ctrl+T

### 导出和打印
- 点击下载图标导出课表为文本文件
- 点击打印图标打印课表

## 🛠️ 技术栈

- **JavaScript** - 交互逻辑和数据处理

## 📱 移动端支持

系统完全响应式设计，支持：
- 手机浏览器访问
- 平板浏览器访问
- 桌面浏览器访问

## 🎯 课程冲突检测

系统会自动检测同一时间段的课程冲突，并提供：
- 粉色渐变背景标记
- "冲突"标签提示
- 柔和的脉冲动画（可关闭）
- 详细的冲突信息展示

## 🔧 自定义配置

### 修改课程数据
编辑 `data.json` 文件，按照以下格式添加课程：

```json
{
  "data": {
    "course": [
      {
        "kchId": "课程编号",
        "name": "课程名称",
        "teacher": "教师信息",
        "time": "上课节次",
        "place": "上课地点",
        "week": "周次",
        "day": "星期"
      }
    ],
    "practice": [
      {
        "courseName": "实践课程名称",
        "kchId": "课程编号",
        "teacherName": "教师信息",
        "grade": "学分",
        "time": "时间安排"
      }
    ]
  }
}
```

### 修改时间配置
在 `index.html` 中修改 `sectionTimes` 对象：

```javascript
const sectionTimes = {
  '1': '08:00-08:45',
  '2': '08:55-09:40',
  // ... 添加更多节次
};
```
